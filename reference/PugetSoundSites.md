# Puget Sound Sampling Locations

A dataset of site coordinates and names for 22 marine field sites in the
Salish Sea.

## Usage

``` r
PugetSoundSites
```

## Format

A data frame with N rows and 3 variables:

- Lon_dd:

  Longitude in decimal degrees

- Lat_dd:

  Latitude in decimal degrees

- Location:

  Site name
